JustGo Audit

A comprehensive design system audit tool for Figma that helps you find layers using variables, styles, and strict layer name matching. Perfect for design system quality assurance and maintenance.

Download Latest Release

Variables & Styles Mode

  • Comprehensive Variable Detection: Find layers using design variables.
  • Complete Style Detection: Locate layers with paint, text, effect, and grid styles.
  • Local & Remote Support: Works with both local tokens and remote library components.

Strict Layer Name Search

  • Exact Name Matching: Find layers by precise layer names only.
  • Component Exclusion: Automatically excludes COMPONENT and INSTANCE types.
  • Predefined Validation: Only allows specific, predefined layer names.

Smart Features

  • Intelligent Scope Selection: Searches selection first, then expands to the page.
  • Detailed Statistics: Get stats on nodes checked and design tokens found.
  • Click to Navigate: Select and zoom to any layer from search results.

Installation for Development

To get started with JustGo Audit for development, simply follow these steps.

  1. Download or clone the repository.
  2. Open the Figma Desktop App.
  3. Go to Plugins → Development → Import plugin from manifest...
  4. Select the manifest.json file from the downloaded folder.
  5. The plugin will appear in your plugins list as "JustGo Audit".
